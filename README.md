# JoanMedia

Personal brand website for Joan Esquivel — Software QA Engineer.

## Pages

- **Home** — Hero, CTAs, latest AI research
- **Conferences** — Conference talks and interviews
- **AI Blog** — AI-generated research articles with categories, series, and sources
- **CV** — Experience, education, certifications, and skills

## Tech Stack

- [Astro 5+](https://astro.build) — Static site generator
- [TailwindCSS v3](https://tailwindcss.com/) — Utility-first CSS
- [DaisyUI v4](https://daisyui.com/) — Component library
- [Sharp](https://sharp.pixelplumbing.com/) — Image optimization

## Getting Started

```bash
pnpm install
pnpm run dev
```

## Build

```bash
pnpm run build
pnpm run preview
```
